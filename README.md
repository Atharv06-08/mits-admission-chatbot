# College Admission Chatbot

This is a College Admission Chatbot developed as a Minor Project assigned by my college (MITS Gwalior).  
The chatbot answers basic questions related to the college admission process.

## Technologies Used
- Python
- Flask

## Features
- Answers admission-related queries
- Simple rule-based chatbot
- Easy to use interface

## How to Run
1. Clone the repository  
2. Install Flask using `pip install flask`  
3. Run the program using `python app.py`  
4. Open browser and go to `http://127.0.0.1:5000/`
